## Requirements
- MATLAB R2024b or later
- Simulink
- Required Toolboxes:
  - Aerospace Toolbox
  - Control System Toolbox
  - Simscape

## How to Run
1. Open `dati_input.m` in MATLAB.
2. Run the script.
3. For Simulink, open `CLASSEA.slx` and click "Run."
